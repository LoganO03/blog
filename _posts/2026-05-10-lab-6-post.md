#### User Stories
As the store owner, I want to be able to list items for sale so that registered customers can buy them.
- Related to stories: 3, 4
- Size: 4

As an unregistered customer, I want to be able to register an account so that I can buy items.
- Related to stories: 3,4
- Size: 6

As a registered customer, I want to be able to start an order so that I can receive items.
- Related to stories: 4
- Size: 5

As a customer, I want to be able to add items to my order so that I can complete my order.
- Related to stories: 3, 5
- Size: 8

As the owner, I want to be able to view finalized orders so that I can get them filled for completion.
- Related to stories: 4
- Size: 2
